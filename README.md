# Danyu Personal Website

A personal portfolio website for Danyu Nie, connecting AI tools, education design, product work, and a living archive of personal exploration.

## Local Preview

```bash
npm run build
npm run dev -- --port 5173
```

Then open:

```text
http://127.0.0.1:5173/
```

## Project Structure

- `site/` - source content, renderer, styles, and interactions
- `public/artifacts/` - public portfolio assets used by the website
- `scripts/` - build, test, and local preview scripts
- `tests/` - lightweight rendering and content tests

## Notes

Large source videos, private resumes, target-company materials, and internal design notes are intentionally excluded from git. The repository currently keeps the lightweight AI projects preview video only; the full source video can be hosted externally later if needed.
